# GitHub 展示关键词与定位

## 一句话定位

中文：用孩子身边的真实物品，生成属于自己的英文毛毡小绘本。

English: Turn everyday objects into a child's own illustrated English storybook.

## GitHub Description

AI-powered photo vocabulary and felt storybook app for children, built with Jetpack Compose and SwiftUI.

## 核心差异点

- 从孩子自己的真实生活出发，而不是使用预置单词卡。
- 把一次识别延伸为单词、插图、故事、朗读和复习的完整闭环。
- 使用统一毛毡视觉与品牌 IP，降低生成内容的割裂感。
- 原生双端实现，保留相机、语音和交互体验的控制力。
