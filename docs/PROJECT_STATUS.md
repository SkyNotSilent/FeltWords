# 项目状态与验证边界

更新日期：2026-06-14

本文用于准确说明当前仓库可以运行到什么程度，避免将“能够编译”“模拟器验证”和“真机完整测试”混为一谈。

## Android

状态：**当前主要可运行版本**

已经验证：

- JDK 17 环境下运行 `./gradlew test assembleDebug` 成功。
- Debug APK 能够构建并安装到 `felt_pixel` Android 模拟器。
- 首页、下拉 IP 场景、拍照页面、识别结果、历史记录、单词本、绘本列表和绘本阅读器完成模拟器截图验收。
- 单词去重、批量撤销恢复顺序、历史图片回填等集合逻辑具备单元测试。
- AI 识别、毛毡图和绘本生成链路已在开发过程中联调。

尚未完成：

- 主流 Android 品牌真机矩阵测试。
- 不同摄像头方向与高分辨率图片内存压力测试。
- 弱网、断网、后台恢复与长时间稳定性测试。
- 正式 Play Store 发布配置与审核。

## iOS

状态：**源码提供，当前版本编译通过，未完成完整运行验收**

已经验证：

- SwiftUI 工程已完整包含在 `ios/`。
- 2026-06-14 使用 Xcode 26 / iOS Simulator SDK 执行 Debug 构建成功。
- 工程包含拍照、识别、历史、单词本、绘本、朗读与本地存储实现。

尚未完成：

- 当前提交版本的完整逐页运行验收。
- iPhone 真机相机、语音与权限测试。
- App Store 签名、归档与发布审核。

## 重要边界

- “编译成功”只代表代码能够生成应用，不代表所有交互和外部服务均已验证。
- “模拟器可运行”不代表相机、扬声器、性能和厂商系统行为已通过真机验证。
- AI 服务可用性依赖本地配置的 API Key、网络与供应商服务状态。
- 正式面向儿童发布前，必须完成隐私、安全、家长同意与数据处理合规工作。
